=========================
How to use my_list.py
=========================
The class is 'MyList'
Only one function called 'print_sorted()'

Import module
=============
>>> MyList = __import__('1-my_list').MyList

Return
======
'MyList.print_sorted()' prints the list with an ascending sort

Tests 'empty list'
-------------------
>>> my_list = MyList()
>>> print(my_list)
[]

>>> my_list = MyList()
>>> my_list.print_sorted()
[]

Tests 'with integers'
---------------------
>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)
>>> print(my_list)
[1, 4, 2, 3, 5]

>>> my_list = MyList()
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]

>>> my_list = MyList()
>>> my_list.append(-1)
>>> my_list.append(-4)
>>> my_list.append(-2)
>>> my_list.append(-3)
>>> my_list.append(-5)
>>> print(my_list)
[-1, -4, -2, -3, -5]
